Netezos is a cross-platform Tezos SDK for .NET developers, simplifying the access and interaction with the Tezos blockchain.

Get Started API References

Cross-Platform

.NET Standard library allows you to use Netezos with big variaty of .NET Frameworks

Simple to Start

It's easy to start interact with Tezos blockchain using the API References

Lightweight

The NuGet package weighs only ~90kB. That means your application won't be overloaded.

Ledger support

Netezos.Ledger allows you to easily interact with Ledger Hardware Wallets (Currently only USB HID for Windows available).

With builtin documentation it's much easier to interact with Tezos blockchain.


Well test covered codebase bla-bla-bla


With .NET Interactive you can use Netezos with any online service that allows you to run Jupyter notebooks
